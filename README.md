# ASTEROIDS: SURVIVOR

Tarayıcıda çalışan roguelike / bullet hell tarzı uzay oyunu (tek HTML dosyası).

## Nasıl oynanır

- **Yerel:** `index.html` dosyasını çift tıklayıp tarayıcıda açın.
- **Canlı (GitHub Pages):** [Oyuna buradan gir](https://mehmeteminkeskin70-lab.github.io/asteroids-survivor/) — İlk yayında 1–2 dakika sürebilir.

## Kontroller

- **WASD** — Hareket  
- **Fare** — Nişan  
- **Space / tık** — Ateş  
- **Shift** — Focus  
- **Q** — EMP  
- **E** — Bomba  
- **ESC** — Duraklat
